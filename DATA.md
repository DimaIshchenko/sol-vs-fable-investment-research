# Data sources, provenance & licensing

Both agents were instructed to build their analysis on **official Ukrainian government statistics**. The raw data each downloaded is preserved untouched inside its model folder (`.../sources/raw/` for Sol, `data/raw/` for Fable) so the work is fully inspectable.

## Where the data comes from

| Source | Provider | Used by |
|---|---|---|
| **Ukrstat / Derzhstat** (SDMX API — enterprise performance, financial results, balance sheet, investment, national accounts, foreign trade) | State Statistics Service of Ukraine | both (statistical backbone) |
| **NBU** (official UAH/USD FX, inflation reports) | National Bank of Ukraine | both |
| **UkraineInvest** (priority sectors, investment guides, incentives) | Ukraine Investment Promotion Office | Sol |
| **World Bank** (GDP, growth, FDI, CPI — cross-checks) | World Bank Open Data | Fable |
| **Legislation** (occupied/hostility territory lists) | Verkhovna Rada / zakon.rada.gov.ua | Sol |
| Secondary / press / consultancy (Interfax, Kyiv Independent, CBRE, KPMG, PwC, PitchBook, etc.) | various | Fable |

Data retrieved on **2026-07-09**. Each model folder carries its own detailed source register — see `.../sources/source_register.csv` (Sol) and `sources/SOURCES.md` + `sources/briefs/` (Fable).

## Licensing & attribution

- The **datasets are not mine and are not relicensed.** They remain the property of their original providers under those providers' terms:
  - **World Bank Open Data** is generally available under **CC-BY 4.0** — attribute the World Bank.
  - **Ukrstat / Derzhstat** and **NBU** publish open government data — attribute the respective agency; do not present derived figures as official.
- This repository's **own contents** — the READMEs, comparison writeups, and each agent's analysis code — are covered by the [LICENSE](./LICENSE) (MIT).
- If you reuse any raw dataset here, go to the original provider for the authoritative, current version and its license.

## Disclaimer

The investment "theses" in this repository were **generated by AI models** as part of a tooling comparison. As documented in [COMPARISON.md](./COMPARISON.md), the final dollar allocations rest largely on **hand-set analyst assumptions**, not on the downloaded statistics. **This is not investment advice.** Nothing here should be used to make real investment decisions.
